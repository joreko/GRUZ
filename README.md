## Возможности

**Работает сейчас:**
- Вставил ссылку → выбрал качество → скачал
- YouTube и [1000+ других сайтов](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md) через yt-dlp
- Выбор формата: видео (mp4, mkv, webm) или только аудио (mp3, m4a, opus, flac)
- Выбор качества от 144p до 4K, выбор FPS, битрейта, кодека
- Очередь загрузок, несколько одновременно
- История загрузок
- Запоминает настройки между сессиями
- Выбор папки сохранения

**В разработке:**
- Обрезка видео по таймкодам прямо в приложении
- Загрузка плейлистов и каналов целиком
- Возобновление прерванных загрузок
- Автообновление yt-dlp
- Уведомления о завершении
- Поддержка прокси
- Автовставка ссылки из буфера обмена при открытии
- Расписание загрузок
- Встроенные субтитры и метаданные в файл
- Поддержка cookies для приватного контента
- Умная память: предпочтения по каналам, определение дублей
- Автогруппировка по каналу в отдельные папки
- Парсинг видео с любого произвольного сайта
- Запись прямых трансляций
- Пресеты личности оркестратора
- API для автоматизации и ИИ-агентов
- Linux и macOS
