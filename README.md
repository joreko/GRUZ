## Возможности

- Вставил ссылку → выбрал качество → скачал
- YouTube и [1000+ других сайтов](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md) через yt-dlp
- Выбор формата: видео со звуком, только видео, только аудио
- Выбор качества от 360p до 4K, FPS, битрейта, видео- и аудиокодека
- Проверка совместимости кодека с контейнером прямо в интерфейсе
- Контейнеры: MP4, MKV, WebM, MOV для видео — MP3, M4A, Opus для аудио
- Очередь загрузок с приоритетами и ручной перестановкой
- Несколько параллельных загрузок
- История загрузок с поиском
- Встраивание субтитров в файл
- Поддержка прокси
- Раздельные папки и шаблоны имён для видео, аудио, плейлистов, Shorts
- Обновление прямо из приложения с возможностью отката
- Запоминает все настройки между сессиями
- Собственный установщик — ярлыки, PATH, ассоциации файлов

**В разработке:**
- Обрезка видео по таймкодам
- Загрузка плейлистов и каналов целиком
- Возобновление прерванных загрузок
- Автообновление yt-dlp
- Уведомления о завершении
- Автовставка ссылки из буфера обмена при открытии
- Расписание загрузок
- Поддержка cookies для приватного контента
- Запись прямых трансляций
- Linux и macOS
